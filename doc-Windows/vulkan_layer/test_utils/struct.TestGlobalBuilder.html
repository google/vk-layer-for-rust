Specify how MockTestLayer should be created.

Can be used to customize the behavior of pre-initialized methods, like InstanceInfo::hooked_commands.

Pre-initialized methods are almost always called, so it is usually needed to call MockTestLayer::set_default_expections right before return to provide a meaningful default behavior.
